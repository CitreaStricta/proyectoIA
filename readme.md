# Proyecto de Inteligencia artificial de generacion procedural de contenido o PCG en la Universidad de Concepcion

## Requerimientos
Este proyecto requiere tener instalado python y las siguientes dependencias:
- Pygame
- scipy
- numpy
## Ejemplo de ejecución
```console
# instalar requerimientos
pip install -r requirements.txt
# ejecutar aplicación
python run.py
```
Si todo funciona correctamente se abrirá una ventana con la ejecución del algortimo, si se desea generar más de un mapa, al finalizar la generación se puede apretar la tecla R para volver a crear un mapa.
